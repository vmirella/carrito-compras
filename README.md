# Marca Perú

Este es un proyecto de catálogo de compra trabajado con Vue. Se consume la api
https://josetello.com/cart, gracias a https://github.com/JoseTelloSolis por la api.
Empecé este desarrollo en 2019.

### Última actualización

Actualmente trabaja con la versión 3 de Vue y Vuetify.

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```
